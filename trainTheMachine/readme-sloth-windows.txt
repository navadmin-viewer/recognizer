To install PyQt4 on windows
Go to https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyqt4
Download PyQt4-4.11.4-cp37-cp37m-win32.whl for 32bit python 3.7
Install with pip install PyQt4...

pip install Pillow
pip install numpy
